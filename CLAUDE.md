# rufios-freaky-friday-todo — Claude Context

## What it is
Deliberately overengineered todo app, built to troll a Twitch streamer (Rufio) from the 100 Devs community by taking her simple project prompt and going completely off the rails. The joke is the overengineering itself.

## Stack
Vanilla JS. No framework. Command pattern, pub/sub event bus, particle effects, analytics, 5-level priority system, full undo/redo (Ctrl+Z/Y).

## Status
Shipped, public. The overengineering is the feature — do not simplify anything.

## Rules
- MORE complexity is correct here, not less
- Every "improvement" should be architecturally absurd given it's a todo app
- The readme tone ("proves you CAN make a simple task manager complex") should be maintained
